## Motivation

Suwwah is a bilingual (Arabic/English) smart tourism assistant for Saudi Arabia.  
We built this project to support Vision 2030 by reducing fragmented travel planning and providing fast, practical guidance. The system generates personalized itineraries, recommends nearby POIs, and recognizes landmarks from user photos to deliver short cultural and historical summaries.

## Features

- Bilingual assistant (Arabic/English)
- Personalized itinerary generation
- Traveler profile support (family, cultural, adventure)
- Real-time POI search via Google Places
- Landmark recognition from photos
- Short cultural/historical summaries
- Basic caching and error handling
- SAND RAG integration
